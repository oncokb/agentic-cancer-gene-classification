First-time guide

This app annotates gene fusions, lets you proofread the generated gene-level results, then exports reviewed data as JSON, CSV, or a Google Sheet.

1. Choose how the model runs

Use Anthropic SDK with an API key, or install a local agent such as Codex, Claude Code, Copilot, or Antigravity.

2. Add required curation access

Paste an OncoKB API token so the pipeline can check whether each gene is already curated in OncoKB.

3. Improve PubMed throughput

Add an NCBI API key to raise PubMed request limits during literature retrieval.

4. Review and export

Run annotations, proofread each gene card, then export to CSV or a Google Sheet for curator review.
